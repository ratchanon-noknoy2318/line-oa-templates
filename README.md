## Overview

A LINE-based healthcare communication system designed to deliver structured messages and automate patient interactions using Flex Messages and webhook integration.

The system focuses on improving communication between healthcare providers and patients through interactive and visually structured messaging.

---

## Problem Statement

Traditional patient communication methods are:

- Manual and time-consuming  
- Limited to plain text messaging  
- Difficult to scale for large patient volumes  
- Lacking structured and interactive formats  

This reduces communication efficiency and clarity.

---

## Solution

This system uses LINE Messaging API and Flex Messages to create a structured and automated communication flow:

- Automated message delivery via webhook
- Structured message layouts using Flex Message
- Interactive UI within chat interface
- Scalable communication for healthcare workflows

---

## Flex Message Design

- Designed 11 structured Flex Message templates  
- Organized information into clear visual blocks  
- Used layout hierarchy for readability  
- Optimized for mobile chat interaction  
- Focused on clarity and quick understanding  

---

## Rich Menu Interface

- Designed 8-grid Rich Menu for navigation  
- Enabled quick access to key actions  
- Reduced need for manual text input  
- Improved user interaction flow  

---

## Messaging Flow

System Trigger → Webhook → Message Processing → Flex Message → User Interaction
