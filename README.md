# Hospital Messaging System

A LINE OA-based hospital messaging system designed to improve patient communication and support healthcare workflow notifications.

## Features
- LINE OA integration for patient communication
- Flex Message templates (11+ message types)
- Rich Menu (8-grid navigation system)
- Automated workflow messaging

## Tech Stack
LINE Messaging API • Node.js • JSON-based message templates

## Impact
- 200+ active users in real healthcare communication workflows

## Real-World Usage
Used in a municipal healthcare communication system for patient interaction and workflow notifications.
